Grid 1: Hardcoded Data (Static) Pass a hardcoded array directly to setData(). The grid displays the data but won't auto-update if the source data changes. Usage: dataSource.setData([...data...]) Grid 2: Reactive Signal Data (Auto-Updates) Pass a signal directly to setData(). The grid automatically detects it's a signal and updates whenever the signal changes—no manual effect needed! Usage: dataSource.setData(signal) — pass the signal, not signal()
💡 Pro Tip: Open the browser console to see events logged. Click the buttons above to update Grid 2's data and watch the grid automatically refresh!
