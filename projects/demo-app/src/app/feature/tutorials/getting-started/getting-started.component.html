Getting started This quick guide shows the minimal structure your application should use once you've installed NGX Kit UI. Keep your AppComponent simple: wrap your app with the library root and add the pieces you need (side menu, top bar, router outlet). App component structure After you've installed the library and imported the modules you need, add the following to your application root template (usually app.component.html): Each application must include the <kit-app-root> as the outermost wrapper. The side menu, top bar and router outlet are optional — include only the pieces your app needs. The side menu can be configured with an initial open state and width. Quick tips One root per app Always keep a single <kit-app-root> per Angular application. This ensures global layout, theme and helpers provided by NGX Kit UI initialize correctly. Optional pieces The <kit-top-bar>, <kit-side-menu> and <kit-router-outlet> are optional. If you don't need a side menu or top bar, omit them — the kit will adapt to the available layout.
