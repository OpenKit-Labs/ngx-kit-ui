Navigation Service The KitNavigationService provides a simple way to manage browser history and a separate view stack for UI-level screens like modals or temporary panels. This keeps your navigation logic clean and predictable. Browser History Use the navigation service to programmatically move back and forward through the user's browser history or to navigate to a new URL. This is useful for creating custom navigation controls or handling form submissions. View Stack The view stack is designed for temporary UI states that shouldn't affect the browser's history. When you push a view, the service uses skipLocationChange to navigate without altering the URL.
