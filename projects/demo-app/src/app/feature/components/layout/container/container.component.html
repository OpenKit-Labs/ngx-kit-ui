Container A responsive container that constrains content width based on the current screen size. When used inside a kit-page, content wrapped in kit-container respects breakpoints, while content placed outside can span the full width. Usage Import the layout module Or you can import the single component Wrap content that should be constrained inside kit-container. Place full-width content (hero sections, banners, etc.) outside of it, directly inside kit-page. Inputs Breakpoints The container applies the following max-width constraints based on screen size. It also centers itself horizontally with margin: 0 auto. Live Demo Resize your browser to see the breakpoint constraints in action. The colored box below is wrapped in kit-container.
