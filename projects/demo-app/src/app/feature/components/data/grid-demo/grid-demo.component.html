Data Grid A simple, extensible data grid for displaying tabular data with pluggable cell renderers. Supports config-driven styling via CSS variables for complete theme customization. Usage Import the data module Or you can import the single component Then use the component in your template Inputs Style Configuration Customize grid styling per-instance using the styleConfig input. All properties are optional and fall back to theme defaults. Available Config Properties Styling Variables Underlying CSS variables used by the grid. Managed automatically via styleConfig.
