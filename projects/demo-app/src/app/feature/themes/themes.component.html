Themes NGX Kit UI comes with a basic set of styles applied to all components out of the box. You can create your own themes by overriding the default styles using CSS variables and using the theme service to apply them Simple Example Lets say you only wanted to change your primary color. You would add this in a new class in your global styles.css file: Then you would use the theme service to apply the new theme class to your application.
